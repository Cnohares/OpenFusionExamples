# 切换编辑器
- 场景编辑器 = alt+q
- 事件编辑器 = alt+w
- 事件列表编辑器 = alt+e
- 全局事件编辑器 = alt+r

# 书签
- 设/消书签 = alt+3
- 上一书签/下一书签 = alt+1/2

# 搜索
- 转至某行 = ctrl+h
- 搜索下一个 = ctrl+g

# 写事件
- 插入新事件组 = alt+n
- 插入新事件 = alt+m

- 插入新条件 = ctrl +n
- 插入新动作 = ctrl + m
- 插入新注释 = ctrl +l
